# PROYECTO-FINAL-AO1
Proyecto de sistema de clasificación de aprendizaje automático para predecir enfermedad cardíaca usando dataset Heart Disease de UCI. Incluye preprocesamiento,análisis exploratorio, entrenamiento y evaluación de modelos supervisados (Regresión Logística, Árbol de Decisión, Random Forest, SVM) con métricas como accuracy, precision, recall y F1-score
